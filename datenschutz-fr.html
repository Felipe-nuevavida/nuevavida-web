Politique de confidentialité

1. Protection des données en un coup d'œil

Remarques générales

Les remarques suivantes donnent un aperçu simple de ce qu'il advient de vos données personnelles lorsque vous visitez ce site web. Les données personnelles sont toutes les données permettant de vous identifier personnellement.

2. Hébergement (GitHub Pages)

Nous hébergeons notre site web chez GitHub Inc., 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA. Lorsque vous visitez notre site web, GitHub collecte les adresses IP et les fichiers journaux afin de garantir la sécurité et le fonctionnement des serveurs. Vous trouverez de plus amples informations dans la politique de confidentialité de GitHub : GitHub Privacy Statement.

3. Contact par e-mail

Si vous nous contactez par e-mail, votre demande, y compris toutes les données personnelles qui en découlent (nom, demande), sera enregistrée et traitée par nos soins aux fins du traitement de votre demande. Nous ne transmettrons pas ces données sans votre consentement.
